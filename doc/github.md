# git started

```sh
git init

git br
git br wei #created a new branch

```

## git config

```sh
git config --global alias.st status
git config --global alias.ci commit
git config --global alias.co checkout
git config --global alias.br branch

git help config

git config -l
alias.co=checkout
alias.br=branch
alias.ci=commit
alias.st=status
alias.lo=log --oneline
```
